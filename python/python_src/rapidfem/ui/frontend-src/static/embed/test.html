Notebook Embed

Embeddable FEM Viewer

Drop simulated antenna / waveguide / line results onto any page with a single script tag.

Default

Static 3D view, no interaction. Useful for documentation thumbnails.

<script src="https://fem.rapidpassives.org/embed/fem-viewer.js"></script>
<fem-viewer
  src="/demo/wr90.json"
  width="100%" height="300px"
></fem-viewer>

Rotating

Continuous camera orbit at default speed.

<fem-viewer
  src="/demo/wr90.json"
  rotate
  width="100%" height="300px"
></fem-viewer>

Cycle — Geometry / Mesh / Field

Animated walk through the three display modes, every ~2 seconds. Same trick the landing cards use.

<fem-viewer
  src="/demo/wr90.json"
  rotate cycle
  width="100%" height="300px"
></fem-viewer>

Interactive

Orbit (left-drag), pan (right-drag), zoom (wheel), double-click to fit.

<fem-viewer
  src="/demo/patch_antenna.json"
  interactive
  width="100%" height="300px"
></fem-viewer>

Field mode

Static field display at a chosen frequency / port index.

<fem-viewer
  src="/demo/iris_filter.json"
  rotate mode="field"
  field-mode="lin" field-freq="20" field-port="0"
  width="100%" height="300px"
></fem-viewer>

Mesh mode

Wireframe view of every named surface — useful for spotting bad mesh density visually.

<fem-viewer
  src="/demo/coax_step.json"
  rotate mode="mesh"
  width="100%" height="300px"
></fem-viewer>

Custom camera preset

Pin the initial view with theta / phi (degrees). Combine with interactive to let the user move from there.

<fem-viewer
  src="/demo/microstrip_line.json"
  interactive theta="60" phi="20"
  width="100%" height="300px"
></fem-viewer>

Transparent background

Blends with whatever the host page uses behind the canvas.

<fem-viewer
  src="/demo/wr90_pml.json"
  rotate transparent
  width="100%" height="300px"
></fem-viewer>
